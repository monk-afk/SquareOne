Commands

For all command parameters: !!! Square brackets, [] required parameters [required] !!! Angled brackets, <> optional parameters <optional> !!! Slash separator, / conditional 'or' <required/condition> [optional/condition] !!! Hyphen separator, - alphanumeric range [0-9/a-z] required conditional ranges 0 through 9 or lowercase 'a' through lowercase 'z'

Category Command Parameter Description
General
/help List commands
/rules Shows the rules
/admin Show the name of the server owner
/privs Show privileges of yourself or another player
/time Show or set time of day
/days Show day count since world creation
/hotbar [1-32] Set hotbar size
/last-login <player_name> Get the last login time of a player or yourself
/when <player_name> Get the last login time of a player or yourself
/who Shows who is online in chat.
/online Shows who is online in chat.
/exp <player_name> Show your or another player's exp data
Chat
/m [player_name] [message] Send a private message to a player
/pm [player_name] [message] Send a private message to a player
/msg [player_name] [message] Send a private message to a player
/fm [message] Faction Chat
/block [player_name] Block player chats, they wont hear you and you won't hear them
/unblock <player_name/!> Unblock player, or clear all, or view block-list
Teleport
/tpr [player_name] Request teleport to player
/tphr [player_name] Request a player teleport to you
/tpy Accept teleport requests from another player
/tpn Deny teleport requests from another player
/tpdnd Block teleport requests from selected or all players
/tpp <place> List or visit the available Public Places.
Location
/spawn Teleport to the spawn point
/home Teleport you to your home point
/home2 Teleport you to your home2 point
/sethome Set your home point
/sethome2 Set your home2 point
/unsethome Restore your last set home pos
/bones Go to last death pos
/fsethome Set the Faction Home
/funsethome Clear the Faction Home
/fhome Go to Faction Home
Economy
/profit-share <join/list> Join Profit Sharing and contribute 10%
/market View market trends
Faction
/fcreate [a-zA-Z0-9] Create new faction. Maximum 9 letters and/or numbers
/fleave Ditch Membership
/finvite [player_name] Send Invite
/fyes Accept Invite
/fno Decline Invite
/fkick [player_name] Expel Member
/fhire [player_name] Hire Member as Staff
/ffire [player_name] Remove Staff Rank
Staff
/warn <player_name> [warning message] In-game public chat warning
/report <player_name> [report message] Silent report to discord
/s [message content] In-game staff channel
Police
/kick [player_name] [0-9mhd] [reason for kick] Kick a player, max 2w, 0 is no timeout
/unkick [player_name] [reason for unkick] Unban a player
/kicks_gui Show XBan GUI
/xban_record [player_name] Show the ban records of a player
/setguardian [player_name] Set or revoke Guardian rank.
/forceblock [player_one] [player_two] Force blocking chat of two players
/mute [player_name] <2-120> <reason for mute> Mutes a player for time in minutes
/unmute [player_name] <reason for unmute> Remove player mute
